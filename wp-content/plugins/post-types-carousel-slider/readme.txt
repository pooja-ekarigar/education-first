Post Carousel & Slider is simple way to build slider and carousel.

== Description ==
This plugin is the easy way for those who wants to convert posts to slider OR carousel. By using this plugin you can display posts OR custom posts as a slider OR carousel. Next and previous post data fetch by Ajax.

It is simple to create post slider OR post carousel for your WordPress site.

View [DEMO](http://aeonian.in/portfolio_page/post-carousel-slider/) for additional information.

The slider uses auto generated shortcodes for making it easier to add it to the custom location.

== Features ==
01. You can create multiple post slider with different options at single page or post
02. Custom post type support
03. Ascending or descending order and Sort retrieved posts
04. Show posts associated with certain categories, taxonomies & tags
05. Show posts based on a keyword search
06. Admin option for update HTML
07. CSS animation [https://daneden.github.io/animate.css/](https://daneden.github.io/animate.css/)
08. Given multiple design option
09. Responsive
10. Admin preview
11. Work with Mousewheel & Touch swipe
12. Infinite Loop
13. Option for return plain html code so you can use other jQuery plugin instead of default ajax functionality (demo available for [bxSlider](http://aeonian.in/no-ajax-bxslider/) & [OwlCarousel](http://aeonian.in/no-ajax-owlcarousel/))
14. You can add fields which created by [Advance custom field](https://wordpress.org/plugins/advanced-custom-fields/) plugin

== Installation ==

= Admin Installer via search =
1. Visit the Add New plugin screen and search for "Post Carousel & Slider".
2. Click the "Install Now" button.
3. Activate the plugin.
4. Navigate to the "Post Carousel & Slider" Menu.

= Admin Installer via zip =
1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select zip file from your computer.
3. Click "Install Now" button.
4. Once done uploading, activate Post Carousel & Slider.

= Manual =
1. Upload the Post Carousel & Slider folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the "Post Carousel & Slider" Menu.
